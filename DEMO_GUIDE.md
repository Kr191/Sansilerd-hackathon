# 🎬 Sovereign AI - Demo Guide

## Quick Start

1. **Start the server** (if not running):
   ```bash
   cd Sansilerd-hackathon
   npm run dev
   ```

2. **Open browser**: http://localhost:3000

## 📱 Demo Flow (5-10 minutes)

### Part 1: Discovery (2-3 min)

1. **Landing Page**
   - Show the clean, modern UI
   - Explain: "AI-driven property investment simulator"

2. **Enter Criteria**
   - Budget: 1,000,000 - 5,000,000 THB
   - Goal: Select "Rent" (for rental income)
   - Location: "Bangkok"
   - Income: 50,000 THB/month
   - Expense: 15,000 THB/month
   - Click "Find My Match →"

3. **View Results**
   - Show loading animation (AI is working!)
   - **Top 3 Matches** appear with:
     - Match scores (85-95%)
     - Property details
     - AI reasons
     - Risk levels
   - Highlight: "The Base Sukhumvit 77" (Rank #1)

### Part 2: Financial Analysis (3-4 min)

4. **Select Property**
   - Click on "The Base Sukhumvit 77"
   - Navigate to Investment Lab

5. **Loan Assessment**
   - Show pre-filled financial data
   - Explain DTI calculation
   - Show max loan capacity: ~1.8M THB
   - Status: "Pre-Qualified" ✓

6. **Investment Simulation**
   - Monthly Payment: ~14,650 THB
   - ROI: 5.8%
   - Net Cash Flow: Show positive/negative
   - Explain: "This is your monthly profit/loss"

7. **Location Trend**
   - Show growth chart
   - 3-year: +12%
   - 5-year: +22%
   - 10-year: +45%

8. **AI Verdict** ⭐ (KILLER FEATURE)
   - Click "🤖 Get AI Verdict"
   - Show loading (AI analyzing...)
   - **Result appears**:
     - ✅ Recommended (or ⚠️ Consider / ❌ Not Recommended)
     - Confidence: 85%
     - AI Insight paragraph
     - Pros (3-5 points)
     - Cons (if any)
   - Explain: "AI analyzed 10+ factors to give this recommendation"

### Part 3: Portfolio (1-2 min)

9. **Navigate to Portfolio**
   - Click "PORTFOLIO" in bottom nav
   - Show portfolio overview:
     - Total Value: 42.8M THB
     - Overall ROI: 18.2%
     - Monthly Cash Flow: 148,200 THB

10. **Smart Alerts**
    - Show "Smart Selling Alert"
    - Explain: "AI monitors market and alerts when profit target is reached"
    - Example: "Sovereign Residences - Unit 402"
    - Target profit +22% reached!

## 🔧 Troubleshooting

### If properties don't load:
- Check console for errors
- Verify API routes are working
- Restart dev server

### If AI Verdict doesn't appear:
- Make sure loan assessment loaded
- Make sure simulation completed
- Check network tab for API errors

### If images don't load:
- Images use Unsplash URLs
- Check internet connection
- Fallback images should load automatically
